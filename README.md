# solution_mock_api

## 获取节目列表
> GET   /menu

| 参数 | 备注 |
| ------------- | ------------- |
| keyword | 搜索关键字 |
| type | 分类id |
| pn | 第几页，从0开始计数 |
| ps | 一页显示数量 |
